# crispr-bioinformatics-project
CRISPR gRNA design Project-bioinformatics Bootcamp

### Tools and website used
-**NCBI** to download target gene sequence .
-**CHOPCHOP** for gRNA design and off-target analysis.
-**SnapGene** for sequence visualization,annotation and map creation.
-**Microsoft Excel** for recording and comparing results .


### STEP BY STEP PROCESS
1. Download the target gene DNA sequence from NCBI website in FASTA format.
2. pasted the sequence into chopchop website and generated multiple gRNA candidates.
3. Selected the best gRNAs based on efficiency,GC content,and lowoff target score.
4. Imported the sequence into Snapgene,annotated the gRNA binding sites and created plasmid maps.
5. compiled all results in microsoft excel and compared them

### Results Summary
 - Selected top gRNAs using chopchop scores.
 - visualized cut sites clearly using snapgene.
